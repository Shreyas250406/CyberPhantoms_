# 🚀 CyberPhantoms - VoIP Monitoring Dashboard

Real-time VoIP call monitoring and analytics dashboard with live call tracking and QoS metrics.

## 🌐 Live Demo
- **Frontend**: [https://cyberphantoms.vercel.app](https://cyberphantoms.vercel.app)
- **Backend API**: [https://cyberphantoms-backend.railway.app](https://cyberphantoms-backend.railway.app)

## 🛠️ Tech Stack
- **Frontend**: React, Recharts, Cytoscape.js
- **Backend**: FastAPI, PyShark, Uvicorn
- **Deployment**: Vercel (Frontend), Railway (Backend)

## 📦 Installation

### Frontend Development
```bash
cd frontend
npm install
npm run dev
